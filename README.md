# Stack-Calculator

## Description
A simple C++ program which read a file containing several mathematical expressions ending with '=' and print the evaluated results into a new file

## Author
Rusiru Anjana Gunaratne

## Supported Operations
| Operation  | Symbol |
| ------------- | ------------- |
| Addition  | +  |
| Substraction  | -  |
| Multiplication  | *  |
| Division  | /  |
| Power  | ^  |
